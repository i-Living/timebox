# TimeBox

Минималистичное PWA для записи на свободные временные слоты — без сервера, без регистрации.

## Как это работает

1. **Организатор** создаёт свободные слоты в календаре и нажимает «Поделиться»
2. Формируется ссылка, в которой зашиты все слоты (сервер не нужен)
3. **Клиент** переходит по ссылке, выбирает время и бронирует
4. Организатор подтверждает бронь — слот помечается занятым

## Возможности

- Создание слотов с групповой вместимостью (capacity > 1)
- Шеринг ссылкой — все данные в URL, никакого бекенда
- Подтверждение / отклонение броней
- Google Calendar синхронизация (OAuth PKCE, при подтверждении создаётся событие)
- ICS-экспорт для Apple/Outlook/Яндекс календарей
- PWA: установка на главный экран, работа офлайн
- Все данные в localStorage — только на вашем устройстве

## Стек

- **Preact** + **TypeScript**
- **Vite** + vite-plugin-pwa
- Без сервера, хостинг на GitHub Pages

## Разработка

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # продакшен-сборка в dist/
```

### Переменные окружения

Создайте `.env` из `.env.example`:

```bash
cp .env.example .env
```

`VITE_GOOGLE_CLIENT_ID` — OAuth Client ID для Google Calendar API (опционально, приложение работает и без него).
