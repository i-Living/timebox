## Wiki Documentation

The project has an auto-generated [LLM Wiki](wiki/) with per-component documentation.

**For the agent:**

1. When working on a component, first check `wiki/entities/` for its documentation page
2. When you add new props, behavior, or components — update or regenerate the wiki
3. To regenerate: `npx react-docgen-ai --src ./src --wiki`

Key files:
- `wiki/index.md` — component catalog
- `wiki/entities/` — per-component pages
- `wiki/log.md` — change history
