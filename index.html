TimeBox

TimeBox — веб-приложение для организаторов групповых занятий (тренировок, мастер-классов, консультаций), позволяющее делиться свободными временными окнами с клиентами. Организатор создаёт временные слоты, формирует ссылку и отправляет её клиенту. Клиент выбирает удобное время, а организатор подтверждает запись. Все данные хранятся локально на устройстве пользователя, без серверного бекенда.

Как TimeBox использует данные Google

TimeBox запрашивает доступ к Google Календарю исключительно для следующих целей:

  • Создание событий — когда клиент подтверждает запись, TimeBox создаёт событие в Google Календаре организатора.
  • Проверка занятости — при создании окон TimeBox может проверять календарь организатора, чтобы исключить уже занятое время.

Данные календаря не передаются третьим лицам, не сохраняются на серверах (приложение работает полностью на устройстве пользователя) и используются только для описанных выше целей.
